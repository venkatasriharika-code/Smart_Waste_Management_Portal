# Smart Waste Management Portal

A full-stack web application designed for smart waste management, tracking, and organization.

## Tech Stack
- **Frontend**: React (built with Vite), TailwindCSS
- **Backend**: Node.js, Express, Prisma, MongoDB

## Project Structure
- `client/` - Contains the React frontend code
- `server/` - Contains the Node.js backend logic and API routes

## How to Run Locally

### 1. Start the Backend Server
First, open your terminal in the `server` directory, install the required dependencies, and run the backend:
```bash
cd server
npm install
npm run dev
```

### 2. Start the Frontend Client
In a new terminal window, navigate to the React client folder, install dependencies, and run the frontend:
```bash
cd client/wastmanagemet
npm install
npm run dev
```
